# Helper

Getting Started
---------------
1. install docker
1. install docker-compose
1. clone repository: `git clone --recursive git@github.com:starting-spark/helper.git`
1. run bootstrap.sh: `./bootstrap.sh`
1. start service: `docker-compose up`
